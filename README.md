# 🎤 AI Interview Prep Platform

An AI-powered interview preparation platform that helps students and job seekers evaluate their technical skills, practice mock interviews, and receive personalized feedback and learning roadmaps.

## 🚀 Features

- 📄 Resume Evaluation
- 🎯 Skill Gap Analysis
- 🛣️ Personalized Learning Roadmap
- 🤖 AI Mock Interview
- 📊 Interview Performance Analysis
- 💾 Session-based Result Storage
- 🌐 Responsive Web Interface

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- AI: Google Gemini API
- Data Processing: Pandas

## 📂 Project Structure

```
frontend/
backend/
README.md
requirements.txt
```

## ⚡ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-interview-prep-platform.git
cd ai-interview-prep-platform
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Start the FastAPI server

```bash
uvicorn main:app --reload
```

4. Open the frontend in your browser.

## 🎯 Purpose

This project is designed to help users prepare for technical interviews by providing AI-driven resume evaluation, mock interviews, performance analysis, and personalized improvement plans.
